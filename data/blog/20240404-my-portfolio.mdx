同樣是一篇紀錄。

繼完成部落格後，接著我完成了個人的 [Portfolio](https://portfolio.kaixsean.com) 頁面，這個頁面是為了宣傳自己用的，不管是為了找工作，或是接案等。
不同於部落格，採用別人的主題來修改。這次跟學習 React/Next.js 這件事綁在一塊，也因為僅僅只有單頁，覺得能降低學習的難易度，不會有一下子太多東西要學習，導致最後還是忘光光。
然而，原本一開始是要用 JavaScript 的，但在處理 metadata 時還是遇到了困難，照 Next.js 做的範例，始終不成功。最後還是改回 Next.js 預設的 TypeScript。也還好，彼此差異不多，其實也沒需要到特別調整什麼。

使用到的資源有以下  
[Next.js 官方學習](https://nextjs.org/learn) CH1 ~ CH5  
[Beginning our React todo list](https://developer.mozilla.org/zh-TW/docs/Learn/Tools_and_testing/Client-side_JavaScript_frameworks/React_todo_list_beginning)

這次製作的頁面是單一頁面的靜態頁面，所以沒有連結資料庫，只用上了 Layouts、Components、Events、Status 以及一些 Metadata 和 SEO相關。
作為入門，我想這樣已是滿足。
後續再繼續邊做邊練習到熟練即可。
